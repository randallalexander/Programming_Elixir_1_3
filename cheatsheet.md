~/.iex.ixs:
  - Elixir load script

iex:
  - Elixir Terminal
  - quit:
    - `ctrl-G` `q`
    - `ctrl-C` `ctrl-C`
  - help:
    - `h`
    - `h <what you want help on>`
  - inspect:
    - `i <value>`
  - import file:
    - `import_file <file name>`

File extensions:
 - .ex -> compiled
 - .exs -> interpreted

Run:
  - scripts:
    - `elixir <file name>.exs`
  - compile and execute:
    - `iex` `c <file name> `
